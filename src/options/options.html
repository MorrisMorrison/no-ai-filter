No-AI Filter

Keywords

One per line. A post/video/result is hidden if its text contains any of these. Plain uppercase entries (AI, AGI, LLM) match case-sensitively with word boundaries, so “email”, “again”, and “chair” won’t trigger. Everything else matches case-insensitively. Power users: wrap an entry in slashes for a raw regex, e.g. /gpt-?[0-9]/i.

Blocked sources

One per line. Hides anything from these sources regardless of wording — subreddits (r/ChatGPT), YouTube channels (@somechannel or the channel name), or link domains (the-decoder.de). Case-insensitive.

No-work mode

Dev/work keywords, one per line — only applied while No-work mode is on. Curated to avoid everyday-word collisions; add aggressive bare terms (React, Go, Rust, code…) at your own false-positive risk.
